## Key features

- **High throughput** — Solana uses a unique Proof of History (PoH) mechanism alongside Proof of Stake to achieve thousands of transactions per second.
- **Low fees** — Transaction fees are a fraction of a cent, making micropayments and high-frequency applications viable.
- **Permissionless** — Anyone can deploy programs, run a validator, or build applications without requiring approval.
- **Multiple clusters** — Use Devnet for development, Testnet for stress testing, and Mainnet Beta for production.
- **Composable programs** — On-chain programs can call one another via cross-program invocations (CPI), enabling rich application composition.
- **Geyser plugins** — Stream account, transaction, and block data to external databases for scalable data indexing.
